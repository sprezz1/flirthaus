Usage Guidelines

Typography Pairing

  • Playfair Display for all headings - adds luxury and sophistication
  • Inter for body text - clean, modern, highly readable
  • Mix serif headings with sans body for perfect balance

Color Usage

  • Rich Black (#121212): Primary brand color, headers, important text
  • Deep Burgundy (#7C1D3C): CTAs, links, accents, premium features
  • Copper (#B87761): Secondary CTAs, hover states, warm highlights
  • Pearl (#F8F4F0): Backgrounds, cards, subtle sections

Spacing Philosophy

  • Generous whitespace creates luxury feel
  • Use 4px base unit for consistency (4, 8, 12, 16, 24, 32...)
  • Sections should have minimum 80px vertical padding

Button Hierarchy

  • Primary (Burgundy): Main CTAs, most important actions
  • Secondary (Copper): Secondary actions, alternative paths
  • Outline: Tertiary actions, less emphasis
  • Ghost: Subtle actions, navigation links
